# 🤖 Automação WEB/LOCAL

Este repositório tem como objetivo centralizar o desenvolvimento de uma solução de **automação para atualização de um programa Java**, executado em ambientes **Web e Local**.

A proposta do projeto é criar um conjunto de ferramentas automatizadas que facilitem o processo de **atualização, manutenção e execução da aplicação**, garantindo mais eficiência, controle e padronização das operações.

---

## 🎯 Objetivo do Projeto

Desenvolver uma automação que permita:

⚙️ Automatizar o processo de atualização da aplicação Java  
🌐 Interagir com sistemas Web quando necessário  
💻 Executar operações em ambiente local do usuário  
📦 Garantir controle e padronização das versões da aplicação  
🚀 Reduzir tarefas manuais no processo de atualização

---

## 🐍 Tecnologias Utilizadas

Todo o desenvolvimento deste projeto será realizado **exclusivamente em Python**.

Principais bibliotecas utilizadas inicialmente:

- 🐍 **Python**
- 📁 **OS** → Manipulação de arquivos, diretórios e processos do sistema operacional
- 🌐 **Selenium** → Automação e interação com aplicações Web

Outras bibliotecas poderão ser adicionadas conforme a evolução do projeto.
